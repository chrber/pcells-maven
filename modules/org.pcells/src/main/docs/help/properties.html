Custumizing the GUI behaviour by manipulating Properties

Properties may be set permanently by using the menu :
Specials -> Properties
Initial values may be chosen using the java '-D' options. 
           java -D<property>=<name> -jar org.pcells.jar
Other variables :
Variable Purpose Values Default
skin Skin Schema soft,7ties,dark,funky,debug OS defaults
histogram.bins Number of bins in Histogram <Integer> > 10 Depends on histogram
histogram.color.bin Color of the histogram bins 0xRRGGBB Black or background color
histogram.color.frame Color of the histogram frame 0xRRGGBB Black
storyboard.background Background picture of the storyboard File name of background picture in pcells directory NONE
storyboard.background.range Range of storyboard background pictures 0,1,2 ... NONE
storyboard.background.scale Should storyboard background picture be scaled into storyboard true,false false
