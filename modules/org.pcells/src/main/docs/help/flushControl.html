Enabling the flush control panel

Before login in, switch to the Setup panel and chose the Modules card. Continuesly click the Next button until it appears disabled. Click Insert. Fill in the Modules Name field with something like Flush Control and fill out the Module Class field with org.dcache.gui.pluggins.flush.JFlushControlPanel. Click the Apply button and Quit the setup. You may loggin. The card layout will have the newly created Flush Control card enabled.

The controls field

Toggle This button allows to switch between the Flush Manager Status Panel and the Flush Storage Class Panels.
Flush Control Manager Name Here you may specify the name of the Flush Manager Cell.
Update This button refreshs all displayed information.
