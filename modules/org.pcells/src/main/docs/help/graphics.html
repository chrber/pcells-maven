Custumizing the GUI Skin

The graphical layout of the this GUI is, honostly spoken, not our major concern. Consequently there is not much you can change or customize about that. Nevertheless :

Colors

In the default setup, background, forground as well as the borders are drawn in two different gray tones. Foreground and background colors may be changed by defining new ones in the OS command line when starting the GUI. 
  like java -D<variable>=<value>
VariableValue
skin.type0=classic;1=border;11=colors
skin.colorsaabbccxxyyzz

Where aabbcc is the hex rgb representation of the background and xxyyzz the hex rgb of the foreground color. If skin.type is set to 1, the color is just used to draw the border, otherwise, having skin.type set to 11, the colors are used to draw most of regular background as well. 

       Example :
        java -Dskin.type=11 -Dskin.colors=123456aabbcc -jar org.pcells.jar
Other variables :
VariablePurposeDefault
histogram.binsNumber of bins in HistogramDepends on histogram
histogram.color.binColor of the histogram binsBlack or background color
histogram.color.frameColor of the histogram frameBlack
storyboard.backgroundBackground picture of the storyboardNONE
The Menu bar Specials -> Skin lets you chose different colors for the current session. Those settings are not persistent.
